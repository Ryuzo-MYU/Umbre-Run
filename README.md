## コンセプト

「傘を使ったゲームをつくる」ことをコンセプトに開発しました。

## システム

怪盗であるプレイヤーは傘を開閉したり向きを変えることができるので、ステージ中に登場するギミックに合わせて傘を動かしてゴールを目指します。

ギミッククリアが遅れると、後ろから追いかけてくる警察に捕まりゲームオーバーとなります。
